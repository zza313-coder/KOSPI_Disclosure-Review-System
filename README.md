📊 KRX KOSPI 공시 의무 종합 검토 시스템

**KRX 유가증권시장 공시규정 및 시행세칙(2026년 최신 개정안)**을 완벽히 반영하여 상장사의 공시 의무 여부를 즉각 판정하고 실무 의견서를 출력하는 웹 시뮬레이터입니다.

🌟 주요 기능

1. 지능형 공시 판단 엔진

대규모법인 자동 판정: 자산총액 2조 원 이상 여부를 자동으로 감지하여 완화된 판단 비율(2.5% 등)을 실시간 적용합니다.

자본잠식 조정 규정 (제3조 제5항): 자기자본이 자본금보다 적을 시, 자동으로 자본금을 자기자본 임계 판단 기준으로 치환하는 정밀 처리를 지원합니다.

2. 재무 데이터 스마트 임포터 (Smart Importer)

원클릭 프리셋: 더블유게임즈의 2025년 결산 공시 데이터(연결/별도)를 클릭 한 번으로 자동 주입합니다.

DART 복사본 텍스트 파싱: 금융감독원 전자공시시스템에서 복사한 원본 텍스트를 붙여넣으면 한글 키워드(자산총계, 매출액, 자기자본, 자본금)를 파싱하여 값을 자동 정제합니다.

배율 자동 감지 (Heuristic): 입력된 숫자의 범위에 따라 '원', '천원', '백만원(DART 표준)' 단위를 자동 판정하여 올바르게 보정합니다.

3. 카테고리별 공시 유형 분석

영업 및 생산활동, 투자 및 자산 활동, 채무 및 재무 구조, 손실 및 재해 등 주요 경영사항 전반을 분석합니다.

달성율을 직관적으로 보여주는 실시간 반응형 게이지 차트를 제공합니다.

4. 실무 의견서 출력 & 제재금 시뮬레이터

의견서 인쇄: 한국거래소 제출용이나 사내 서면 검토 의견서 포맷에 맞춘 깔끔한 출력용 리포트를 생성하며 프린트 기능(Print)을 완벽히 지원합니다.

시행세칙 [별표 2] 제재금 시뮬레이터: 공시 지연이나 오류 위반 발생 시 위반 동기(고의/과실) 및 중요도에 따른 예상 벌점과 추가 위반제재금을 모의 산정합니다.
